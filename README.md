# ⚙️ Jenkins Task Project

## 📖 Overview
This project demonstrates how to use Jenkins for automating build and deployment workflows using CI/CD concepts. It shows how Jenkins integrates with GitHub repositories to automate tasks.

---

## 🚀 What I Did
- Installed and configured Jenkins locally
- Connected Jenkins with GitHub repository
- Created jobs/pipelines for automation
- Triggered builds automatically on code changes
- Learned CI/CD workflow basics
- Executed build and deployment tasks using Jenkins pipelines

---

## 🛠️ Technologies Used
- Jenkins
- Git & GitHub
- Docker (if used in your project)
- CI/CD Pipeline concepts

---

## ⚙️ How It Works
1. Code is pushed to GitHub repository  
2. Jenkins detects changes (webhook or manual trigger)  
3. Jenkins pulls latest code  
4. Executes defined pipeline steps  
5. Builds / tests / deploys application
